# Data Exploration and Cleaning using Pandas

## Objective
The objective of this assignment is to learn Python basics and perform basic data exploration and data cleaning using the Pandas library.

## Dataset
- Dataset Name: Sample - Superstore.csv

## Tasks Performed

1. Imported the Pandas library.
2. Loaded the CSV dataset into a Pandas DataFrame.
3. Explored the dataset using:
   - head()
   - tail()
   - shape
   - columns
   - dtypes
4. Checked for missing values using:
   - isnull().sum()
5. Filled missing values where required.
6. Filtered rows based on Sales values.
7. Selected specific columns for analysis.
8. Removed duplicate records.
9. Created a new derived column named **total_amount**.
10. Saved the cleaned dataset as **cleaned_superstore.csv**.

The dataset was successfully explored, cleaned, and processed using Pandas. A cleaned version of the dataset was created and saved for further analysis.
